# QR-код сайта ЮМИ

## Куда ведёт QR

Сканирование открывает короткий стабильный адрес:

- **Кириллица:** `https://юми-корм.рф/qr`
- **Punycode:** `https://xn----otbhfdlo0i.xn--p1ai/qr`

Страница `/qr` на сайте перенаправляет на главную с UTM-метками:

`/?utm_source=offline&utm_medium=qr&utm_campaign=launch_first_clients`

Так можно отслеживать переходы с печатных материалов, визиток и упаковки.

## Какой файл использовать

| Файл | Назначение |
|------|------------|
| `yumi-main.png` | **Печать** — визитки, флаеры, наклейки, постеры (2048×2048 px) |
| `yumi-main.svg` | **Быстрые отправки** — мессенджеры, презентации, макеты (вектор, масштабируется без потери) |

## Перед печатью

1. Отсканируйте QR с телефона (камера или приложение-сканер).
2. Убедитесь, что открывается сайт и вы попадаете на главную.
3. Проверьте на светлом и тёмном фоне макета — контраст должен быть высоким.

## Минимальный размер печати

Рекомендуется **не меньше 30×30 мм** на готовом макете. Меньший размер может ухудшить считывание.

## Перегенерация

```bash
npm run qr:generate
```
