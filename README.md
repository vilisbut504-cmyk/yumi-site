# ЮМИ — сайт бренда

Премиальные натуральные сушёные лакомства и жевательные продукты для собак.
Демо-версия интернет-магазина: каталог, карточки товаров, подбор лакомства,
корзина и оформление заказа без онлайн-оплаты.

Фото товаров на этом этапе хранятся локально в `public/products` (`.webp`).

## Публичная ссылка

**https://vilisbut504-cmyk.github.io/yumi-site/**

## Локально

```bash
npm install
npm run dev
```

## Обновить сайт на GitHub Pages

После правок в коде:

```bash
npm run deploy:pages
```

Скрипт соберёт статическую версию и отправит её в ветку `gh-pages`.
