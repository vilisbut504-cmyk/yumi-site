# ЮМИ — концепция сайта бренда

Премиальный лендинг сухого корма для собак. Демо-версия для согласования концепции.

## Публичная ссылка

**https://vilisbut504-cmyk.github.io/yumi-site/**

## Локально

```bash
npm install
npm run dev
```

## Обновить сайт на GitHub Pages

После правок в коде:

```bash
npm run deploy:pages
```

Скрипт соберёт статическую версию и отправит её в ветку `gh-pages`.

## Стек

Next.js 15 · TypeScript · CSS (globals.css)
